# Weather App With Geolocation

## **** USED ****
* *** HTML5 ***
* *** CSS3 ***
* *** JavaScript[ES6] ***
* *** Geolocation API ***
* *** OpenWeatherMap API ***

## Description
A simple weather app that uses the Geolocation API to get the user's current location and display the current weather.

## Features
* Get current location using Geolocation API
* Display current weather conditions (temperature, description, etc.)

## Installation
1. Clone the repository
2. Replace `YOUR_API_KEY` in `script.js` with your actual Open-meto API key
3. Open `index.html` in a web browser

## Usage
1. Allow the app to access your location
2. View the current weather conditions for your location
